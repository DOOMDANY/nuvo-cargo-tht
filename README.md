## Getting Started

* Install dependencies:
	```bash
	nvm use
	npm i
	```
	If nvm is not installed use the node version set in `.nvmrc` file
* Run the development server:
	```bash
	npm run dev
	```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
